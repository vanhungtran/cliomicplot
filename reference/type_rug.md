# Rug Plot Type

Adds rug marks (small tick lines) to indicate data density along the x
and/or y axes.

## Usage

``` r
type_rug(sides = "b", alpha = 0.5, linewidth = 0.3)
```

## Arguments

- sides:

  Which sides to draw rugs on: "b" (both), "l" (left/bottom), "t" (top),
  "r" (right), or combinations like "bl", "tr"

- alpha:

  Rug transparency

- linewidth:

  Rug line width

## Value

A cliplot_type object
