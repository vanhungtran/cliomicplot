# Lollipop Chart Type

Creates stylish lollipop charts — a modern alternative to bar charts
with circle endpoints on thin stems.

## Usage

``` r
type_lollipop(
  point_size = 4,
  stem_width = 0.8,
  stem_color = "grey60",
  point_color = NULL
)
```

## Arguments

- point_size:

  Circle size (default 4)

- stem_width:

  Stem line width (default 0.8)

- stem_color:

  Stem line color (default "grey60")

- point_color:

  Circle fill color

## Value

A cliplot_type object
