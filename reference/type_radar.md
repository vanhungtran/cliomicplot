# Radar / Spider Chart Type

Creates stunning radar (spider) charts for multivariate comparisons
across groups. Perfect for profile visualization.

## Usage

``` r
type_radar(
  alpha = 0.25,
  linewidth = 1,
  gridline_width = 0.3,
  show_legend = TRUE
)
```

## Arguments

- alpha:

  Fill transparency (default 0.25)

- linewidth:

  Line width (default 1)

- gridline_width:

  Grid line width (default 0.3)

- show_legend:

  Show legend (default TRUE)

## Value

A cliplot_type object
