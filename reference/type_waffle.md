# Waffle Chart Type

Creates stunning waffle (square pie) charts for proportional data. Each
square represents a unit, making proportions intuitive.

## Usage

``` r
type_waffle(rows = 10, colors = NULL, size = 1.5, pad = 0.1)
```

## Arguments

- rows:

  Number of rows in the waffle grid (default 10)

- colors:

  Vector of fill colors

- size:

  Square size (default 1.5)

- pad:

  Padding between squares (default 0.1)

## Value

A cliplot_type object
