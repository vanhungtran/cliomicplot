# Circular Barplot Type

Creates eye-catching circular barplots where bars radiate from the
center. Perfect for ranking visualizations with many categories and a
dramatic visual impact.

## Usage

``` r
type_circular_bar(
  bar_width = 0.8,
  inner_radius = 0.3,
  show_labels = TRUE,
  label_size = 3
)
```

## Arguments

- bar_width:

  Width of bars in degrees (default 0.8)

- inner_radius:

  Inner radius of the circle (default 0.3)

- show_labels:

  Show bar labels (default TRUE)

- label_size:

  Label text size (default 3)

## Value

A cliplot_type object
