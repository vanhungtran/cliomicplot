# Dumbbell Plot Type

Creates elegant dumbbell charts for comparing two time points or
conditions. Shows change with connected points and magnitude.

## Usage

``` r
type_dumbbell(
  point_size = 3,
  line_width = 0.8,
  color_first = "#4DBBD5",
  color_second = "#E64B35"
)
```

## Arguments

- point_size:

  Point size (default 3)

- line_width:

  Connecting line width (default 0.8)

- color_first:

  Color for first time point

- color_second:

  Color for second time point

## Value

A cliplot_type object
